# Packet Tracer Files - Lab Assignment 1

This folder contains all Cisco Packet Tracer simulation files for ENCS304 Lab Assignment 1.

## Files Included

- Task 1 CN_Lab.pkt  
  → Star Topology using 2960 Switch

- Task 2 CN Lab.pkt  
  → Bus-like Topology using Hub

- Task 3 CN Lab.pkt  
  → Ring-like Topology using 3 Switches (STP Enabled)

- Task 4 CN Lab.pkt  
  → Link Failure Test (STP Reconvergence)

All devices are configured within the 192.168.10.0/24 subnet.
Connectivity was verified using ICMP ping tests.
